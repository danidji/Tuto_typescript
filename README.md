# Tuto Typescript 

## ligne de commande

### Installation

ATTENTION ne fonctionne pas avec yarn

```bash
npm i -g typescript
```

### Compilation d'un fichier TS en JS 

```bash 
tsc +<nom du fichier>
```

### Création du fichier tsconfig.json 
=> préciser l'utilisation ??!

```bash
tsc --init
```

### Log des erreurs 

````bash 
tsc -w
```
